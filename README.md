## 更新了第一个版本
1、sd一定要打开api的

2、设置sd的地址，如果是本地就设置http://127.0.0.1:6006  注意端口要一定要正确

3、设置gpt的key ，需要科学上网

4、设置一个本地存图片的路径


## 工作流程
1、添加小说

2、自动根据换行符拆分词

3、自动使用GPT去解读场景转化成sd的描述词

4、自动根据描述词去画图

5、图片会保存在本地的路径
